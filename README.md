# Lifestyle Data Analysis (R)

##  Overview
This project uses **R** to explore lifestyle data, focusing on correlations between daily habits and health/productivity outcomes.

##  Library Stack
- **tidyverse:** For data manipulation (`dplyr`) and visualization (`ggplot2`).
- **lubridate:** For handling time-series lifestyle data.

## 📈 Key Analysis
1. **Data Cleaning:** Handling missing values and normalizing habit frequencies.
2. **Visualizations:** Distribution of lifestyle scores across demographics.
3. **Statistical Testing:** Correlation analysis between sleep, exercise, and stress levels.
